「帮我选」Help Me Choose
============

- 一个帮助选择困难症女友选择吃什么的网站

- 开发背景：
Jasin Yip的女友经常苦恼吃什么东西，所以作为程序猿的Jasin Yip开始开发一个帮她选择吃什么的网站。

- 应用区域：
广东轻工职业技术学院南海校区
